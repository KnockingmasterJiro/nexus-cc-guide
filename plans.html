そもそもプランって何？

Claude Code を使うには有料プランへの加入が必要です。プランによって「使える量」と「機能」が変わります

ざっくり言うと： Claude Code は「AIに仕事を手伝ってもらうサービス」です。プランは「月額いくら払うか」の選択肢。高いプランほどたくさん使えます。スマホの通信プランみたいなものだと思ってください。

  • claude.ai でチャットできる
  • 使える回数が少ない

Claude Code が使える最安プラン。「まず試してみたい」ならこれ。

  • Claude Code が使える
  • claude.ai も使える
  • 使用量：基準（1x）
  • 管理機能なし（個人利用）

$100〜200 / 月
約 15,000〜30,000円 / 月

ガッツリ使う人向け。使用量がProの5倍〜20倍になる。翁が使っているのはこれ。

  • Claude Code が使える
  • 使用量：5x（$100）/ 20x（$200）
  • 一番賢いモデルが使い放題
  • 管理機能なし（個人利用）

「使用量」って何？ Claude Code に指示を出すと、裏側で「トークン」というものを消費します。使用量が多い = 1日にたくさん指示を出せる、ということ。少ないプランでも使えますが、たくさん使うと「今日はここまでです」と制限がかかります。翌日にはリセットされます。
Team の「Standard席」と「Premium席」って？ 同じチーム内でも、人によって席の種類を分けられます。たまにしか使わない人は Standard（安い）、ガッツリ使う人は Premium（使用量が多い）、というように混在OK。全員同じ席にする必要はありません。
3

結局、どっちに入ればいい？

チームの状況に合わせて選ぼう

👤 個人プラン（Pro / Max）がいいケース

  • 使う人が5人未満 — Team プランは最低5人なので、少人数なら個人プラン一択
  • まずは試したい — Pro（$20）で始めて、本格的に使うなら後からTeamに変更できる
  • 各自で自由に使いたい — 管理者を置かず、それぞれが好きに契約・解約できる
  • コスト重視 — Pro $20 が最安。月3,000円で Claude Code が使える

👥 チームプランがいいケース

  • 5人以上で使う — 管理者がメンバーの追加・削除を一括管理できて便利
  • 会社の経費で落としたい — 請求がまとまるので、経理処理がラク
  • 入退社の管理をしたい — 辞めた人のアカウントをすぐ停止できる
  • プロジェクトを共有したい — チーム内でナレッジを共有する機能がある
  • セキュリティが気になる — SSO（シングルサインオン）やドメイン認証に対応
おすすめの進め方： まだ誰も使ったことがないなら、まず 個人Pro（$20/月）で1〜2人が試す → 良さそうなら チームプランに移行、が一番リスクが少ないです。個人プランの契約は後からいつでもキャンセルできます。
注意： 個人アカウントからチームプランに「アップグレード」しても、個人アカウントのデータ（会話履歴など）はチームに引き継がれません。個人の有料契約は別途キャンセルが必要です。

契約手順

個人プラン / チームプラン、それぞれの入り方

👤 個人プラン（Pro）の契約手順

1

claude.ai にアクセスしてアカウント作成

ブラウザで claude.ai を開き、メールアドレスを入力。Google アカウントでの登録も可能です。メール認証を完了させてください。

2

設定画面から「Upgrade plan」

ログイン後、画面左下にある自分の名前（またはイニシャル）をクリック → 設定メニューの「Billing」→「Upgrade plan」をクリック。

直接アクセスもOK： claude.ai/upgrade をブラウザで開いても同じ画面に行けます。
3

「Pro plan」を選択して支払い情報を入力

「Get Pro plan」をクリック → 月払い or 年払いを選択 → クレジットカード情報を入力 →「Subscribe」で完了。

これで Claude Code が使えるようになります。あとは導入ガイドのセットアップ手順に従ってインストールしてください。

👥 チームプランの契約手順（管理者向け）

1

会社のメールアドレスで claude.ai にログイン

claude.ai にアクセスし、会社のメールアドレスでアカウントを作成。Gmail や Yahoo などのフリーメールでは Team プランを契約できません。

重要： @gmail.com, @yahoo.co.jp などの個人メールでは Team プランに入れません。必ず会社ドメインのメールアドレス（例: @nexus.co.jp）を使ってください。
2

Team プランを選択して組織を作成

ログイン後、claude.ai/upgrade にアクセスして「Team」プランを選択。組織名を入力し、支払い情報を登録します。

Standard席（$25〜30/人）と Premium席（$100〜150/人）を組み合わせて席数を決めます。最低5席から。

3

メンバーを招待する

組織ができたら、メンバーを招待します。方法は3つ：

  • メールで招待 — 設定 → Organization →「Add member」→ メールアドレスを入力。席の種類（Standard / Premium）とロールを指定
  • 招待リンクで招待 — 共有リンクを発行して、Slack やメールで配布。リンクをクリックした人が自動で参加できる
  • メンバー同士の招待 — 管理者でなくても、既存メンバーが同僚をメール招待できる
注意： 招待を送った時点で席が1つ消費されます（相手がまだ参加していなくても）。席数の管理に気をつけてください。
4

メンバーが参加 → Claude Code を使い始める

招待されたメンバーは、メールのリンクをクリックして組織に参加 → あとは導入ガイドのセットアップ手順に従って Claude Code をインストールするだけ。

管理者は「設定 → Organization」からメンバーの追加・削除・席の変更がいつでもできます。

5

よくある質問

プラン選びで迷ったらここを見てください

無料プランでは Claude Code は使えないの？ ▼
はい、使えません。Claude Code を使うには Pro プラン（$20/月）以上への加入が必要です。無料プランで使えるのは claude.ai のWebチャットだけです。
Pro（$20）で足りる？ Max にした方がいい？ ▼
「1日に数回の指示」程度なら Pro で十分です。ただし、長時間ガッツリ開発する場合は使用量の上限に達しやすく、「今日はここまで」と止まることがあります。その場合は Max（$100〜）にすると快適になります。まずは Pro で始めて、制限に引っかかるようなら Max に上げるのがおすすめです。
Team プランは 5人以下でも入れる？ ▼
いいえ、Team プランは最低5人（5席）からです。4人以下の場合は各自で個人プラン（Pro や Max）に加入してください。
Team で Standard席と Premium席を混ぜられる？ ▼
はい、混在できます。たとえば「Claude Code をガッツリ使う人は Premium、たまに使う人は Standard」というように、人ごとに席の種類を変えられます。あとから変更もできます。
個人プランから Team に乗り換えたら、今までのデータは引き継がれる？ ▼
いいえ、引き継がれません。個人アカウントと Team 組織は別扱いです。Team に参加した後、個人の有料サブスクリプションは別途キャンセルする必要があります（自動では解約されません）。
会社のフリーメール（Gmail）で Team プランに入れる？ ▼
いいえ、Team プランの管理者アカウントには会社ドメインのメールアドレスが必要です（@gmail.com や @yahoo.co.jp は不可）。ただし、管理者がメンバーを招待する際に異なるドメインのメールアドレスを許可する設定もあります。
支払いはドル建て？ 日本円では払えない？ ▼
基本はドル建て（USD）ですが、地域によって表示通貨が異なる場合があります。日本のクレジットカードで問題なく支払えますが、為替レートとカード会社の外貨手数料がかかります。claude.ai/upgrade にアクセスすると、あなたの地域での価格が表示されます。
いつでも解約できる？ ▼
はい、いつでも解約できます。解約しても、残りの期間（月末まで）は引き続き利用可能です。年払いの場合も途中解約は可能ですが、日割り返金の扱いについては公式ヘルプを確認してください。
