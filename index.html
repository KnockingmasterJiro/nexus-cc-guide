Claude Code とは？

PCの「黒い画面」でAIと会話しながら、ツールやアプリを作れるサービス

特徴

  • AIとチャットする感覚で使える — PCの「黒い画面」（ターミナルと呼びます）で claude と打つだけで起動。LINEで会話するように、日本語で指示を出せます
  • エンジニアでなくても使える — 翁（キング）はエンジニアではないが、Claude Codeだけで6つの業務ツールを開発。プログラミングの知識ゼロでOK
  • フォルダの中身を丸ごと理解 — 既にあるファイルを読み込んで、文脈を理解した上で修正・追加ができる
  • エラーが出てもコピペでOK — エラーメッセージをそのまま貼り付ければ、原因を特定して直してくれる
  • Webアプリ、自動化、データ分析… なんでも作れる。「何を使えばいいか」もAIが選んでくれる
非エンジニアの実績: SNS攻略事業部の翁が、約2週間で6つの業務自動化ツールを構築。毎日の定常業務を大幅に効率化しました。

セットアップガイド

完全初心者でもOK — 手順通りにやれば使えます

0

必要なもの

  • Windows 10 または 11 のPC
  • インターネット接続
  • Claudeのアカウント（Max プラン $100/月 または Team プラン推奨）
Claude Max プランとは、月額 $100 で Claude Code が使い放題になるプランです。個人利用ならこれでOK。チームで使うなら Team プランも検討してください。
1

「ターミナル」って何？ — まず知っておこう

ターミナル = PCに文字で指示を出す画面のことです。

LINEでメッセージを送るように、PCにコマンド（命令文）を打ち込んで操作します。
Claude Code はこのターミナル上で動きます。難しそうに見えますが、基本はコピペして貼り付けるだけなので安心してください。

ターミナルの開き方:

  1. キーボードの Windows キー（左下の旗マーク）を押す
  2. そのまま「powershell」と入力して検索する
  3. 「Windows PowerShell」が出てくるのでクリック
  4. 黒っぽい画面が出ればOK！ これがターミナルです
Windows 11 の場合: 「ターミナル」と検索すると「Windows Terminal」が出てきます。こちらでもOK。中身は同じです。
← こういう黒い画面が出ます（例）

PS C:\Users\あなたの名前> ← ここにコマンドを打ち込みます
「PS C:\Users\...>」って何？ — これは「今ここにいますよ」という表示です。自分で入力するものではありません。この後ろにコマンドを打ちます。
2

Node.js と Git をインストール

Claude Code を動かすのに必要なソフト2つを、コマンドだけでインストールします。

ターミナル（Step 1で開いた画面）に、以下を1行ずつコピペして Enter:

# ① Node.js をインストール（Claude Code の土台）
winget install OpenJS.NodeJS.LTS

# ② Git をインストール（ファイルの変更履歴を管理するツール）
winget install Git.Git

それぞれ「インストールしますか？」と聞かれたら Y を押して Enter。数分で完了します。

winget って何？ — Windows に最初から入っている「アプリストア」のコマンド版です。ソフトのダウンロード〜インストールを全部自動でやってくれます。

インストールが終わったら、ターミナルを一度閉じて開き直してください（これ重要！）。
その後、以下をコピペして確認:

# バージョンが表示されればOK
node --version
git --version

v22.x.x や git version 2.x.x のように表示されれば成功です。

Node.js って何？ — Claude Code を動かすための土台ソフトです。
Git って何？ — 「いつ・何を変えたか」の記録を自動で残してくれるツールです。間違えても元に戻せるので安心。Claude Code が裏側で自動的に使っています。
3

Claude Code をインストール

ターミナルに以下をコピペして Enter:

npm install -g @anthropic-ai/claude-code

文字がたくさん流れますが、そのまま待ってください。1〜2分で完了します。

npm って何？ — Node.js を入れたときに一緒にインストールされた「アプリストア」のようなものです。ここからClaude Code をダウンロードしています。
4

作業フォルダを作って、Claude Code を起動

Claude Code はフォルダの中で動きます。まず作業用のフォルダを用意しましょう。

ターミナルに以下を1行ずつコピペして Enter:

# ① デスクトップに「my-project」フォルダを作る
mkdir ~/Desktop/my-project

# ② そのフォルダに移動する
cd ~/Desktop/my-project
mkdir = フォルダを新しく作るコマンド。cd = そのフォルダに移動するコマンド。
デスクトップに「my-project」というフォルダが出来ているはずです。確認してみてください。

そして、いよいよ起動:

claude

初回だけ、ブラウザが自動で開いてログイン画面が表示されます。
Claudeのアカウントでログインすれば認証完了。ターミナルに戻ると Claude Code が使えるようになっています。

5

最初のお試し — 「おみくじアプリを作って」

Claude Code が起動すると、入力待ちの状態になります。
試しにこう入力して Enter を押してみてください:

おみくじアプリを作って

すると Claude が自動でコードを書いて、HTMLファイルを作ってくれます。
デスクトップの「my-project」フォルダの中にファイルが出来ているので、ダブルクリックで開くとおみくじアプリが動きます。

これが基本の流れです:
① やりたいことを日本語で伝える → ② Claude が作る → ③ 動かして確認
この繰り返しで、どんどんツールが出来上がっていきます。
6

CLAUDE.md — 「マイルール」を設定しよう（慣れてきたら）

作業フォルダの中に CLAUDE.md というファイルを作っておくと、Claude に「毎回こういうルールで動いてね」と指示できます。

Claude Code に直接お願いすれば作ってくれます:

CLAUDE.md を作って。以下のルールを書いて:
- 日本語で対応すること
- わかりやすくコメントを入れること

毎回同じことを言わなくても、Claude が覚えてくれるようになります。

!

使う前に知っておいてほしいこと

Claude Code は強力なツールです。だからこそ、これだけは守ってください

⚠ アカウント情報（ID/パスワード）を安易に渡さない

Claude Code にログインIDやパスワードを教えると、そのアカウントで出来ることは何でも出来てしまいます。便利な反面、操作ミスや意図しない動作で取り返しのつかないことが起きる可能性があります。

✗ 特に危険な例:
・会社の Google アカウント → メール・ドライブ・スプシ全部にアクセスできてしまう
・管理画面のフル権限アカウント（例: adnex） → 設定変更・データ削除が可能になる

✓ 安全なやり方:
・権限を絞った専用アカウントを作って、それを使う
・「閲覧のみ」「編集のみ」など、必要最低限の権限だけ付ける
・パスワードを渡す場合も、フル権限ではなく限定権限のアカウントで

ルール: フル権限のアカウント情報は渡さない。必ず専用アカウントを作る。

👀「はい」を押す前に、何をしようとしているか確認する

Claude Code はファイルの編集やコマンドの実行をする前に、「これやっていい？」と確認を求めてきます。これが安全装置です。

  • 表示された内容がよく分からないときは「n（いいえ）」を押して、「今何しようとした？日本語で説明して」と聞き直してください
  • 特に「rm」「delete」「drop」のような単語が見えたら要注意（= 削除系のコマンド）
  • 焦って全部「y」で進めない。1つずつ確認する癖をつけましょう

🔒 入力した内容はクラウド（外部サーバー）に送信される

Claude Code に入力した文字は、AIの処理のためにAnthropic（アメリカのAI企業）のサーバーに送信されます。

  • 社外秘の数字やデータをそのまま大量に貼り付けるのは避ける
  • 個人情報（お客様の名前・電話番号など）は入力しない
  • 「このデータ分析して」ではなく、ダミーデータで動作確認してから本番データを使う

📁 必ず専用フォルダの中で作業する

Claude Code は起動したフォルダの中のファイルを読み書きします。デスクトップや「ドキュメント」フォルダで直接起動すると、関係ないファイルを触ってしまう可能性があります。

必ず Step 4 で作ったような専用フォルダの中で起動してください。

5

使い方のコツ

Claude Code をうまく使いこなすためのヒント

日本語でOK

プログラミング用語を知らなくても大丈夫。やりたいことをそのまま日本語で伝えてください。

「ボタンを押したらデータをダウンロードできるようにして」

具体的に伝える

「いい感じにして」より「こういう条件で」と具体的に伝えるほど精度が上がります。

「売上とクリック数からCPCを計算して、グラフも出して」

条件は箇条書きで

複数の要件があるときは番号付きで整理すると、漏れなく対応してくれます。

「画面を作って。条件:
① ダークテーマ
② レスポンシブ対応
③ ログイン機能付き」

エラーはコピペ

エラーが出たら、エラーメッセージをそのままコピーして Claude に貼り付ければ直してくれます。

「このエラーが出た:
Error: Cannot find module 'express'」

困ったら聞く

「何ができる？」「どうやって〜する？」と聞けば、できることを教えてくれます。

「スプレッドシートを自動操作する方法ある？」

/help でヘルプ

Claude Code 内で /help と入力すると、使えるコマンド一覧が表示されます。

/help
/clear（会話リセット）
/compact（要約して続行）

最後に一番大事なこと

ここまで色々書きましたが、正直Claude Code の使い方を勉強する必要はありません。
とにかく使ってみてください。
分からないことがあったら、Claude Code 自身に「これどうやるの？」と聞けばいいんです。
使い方を教えてくれるのも、Claude Code です。

まずは触る。それが一番の近道です。
